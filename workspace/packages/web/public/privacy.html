Privacy Policy

Effective date: January 27, 2026

Summary

Contractor Passport helps contractors share verified signals (e.g., GitHub and LinkedIn OAuth) with employers. We collect only the data needed to provide verification and issue credentials.

Information We Collect

  • Account details: name, email, and contractor identifier.
  • Verification data: OAuth profile data from GitHub/LinkedIn (e.g., user id, profile URL, verified email).
  • Credential metadata: issuance time, tier, and verification status.
  • Usage logs: IP address, timestamps, and actions for security and audit.

How We Use Information

  • To run verification flows and issue credentials.
  • To prevent abuse and keep the system secure.
  • To improve reliability and debug issues.

Sharing

We share verification results only with parties the contractor explicitly provides a verification link to. We do not sell personal data.

Retention

We retain verification data and audit logs as needed to operate the service and comply with legal obligations. Contractors may request deletion by contacting us.

Security

We use industry-standard controls (rate limiting, audit logs, signed tokens) to protect data. No system is 100% secure; we continuously improve protections.

Contact

For privacy requests, email: privacy@contractorpassport.com
